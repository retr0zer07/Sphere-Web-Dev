# Sphere Dev Web

Landing page for Sphere Dev Web (single-page marketing site).

## Deploy

1. Upload the repository content to your static host root.
2. Ensure `index.html` is served from `https://spheredevweb.com/`.
3. Keep `sitemap.xml` and `robots.txt` at the site root.
4. Optional: configure analytics providers (`window.dataLayer` and/or `window.plausible`).

## Local preview

Open `/home/runner/work/Sphere-Web-Dev/Sphere-Web-Dev/index.html` directly in the browser (`file://`) or serve with any static server.
